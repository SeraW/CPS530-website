Materialize

  • devicesA CSS Framework Made by Google
    Materalize is mostly used in the United States. According to the State of CSS report 2019, 72% of those surveyed had heard of Materialize, and 37% were interested in using it.

    It was overall ranked 3rd in terms of popularity -- being beaten out by Bootstrap and Foundation. However, only 17.1% of developers say they would use it again, compared to 45% for Bootstrap.
  • paletteEasy and Efficient
    The built-in custom components make for easy and fast development. There are many free plugins for drop-downs, collapses and more. The framework is also very responsive, and makes for a great user experience.

    Overall, Materalize is easy to pick up, with great built-in features, and ready-to-use page elements.

  • vpn_lockNew, Small, Limited

    Materialize does not work on older browsers, such as anything older than Internet Explorer 10. In addition, the framework has a smaller community with less detailed documentation and forums.

    Designs are also fairly limited; websites that you create may all act similarly in terms of UI.

  • group_addA Small Community

    Companies such as DroneDeploy, IWB, and Greencode use Materialize. Materialize is used for a variety of different sites; mostly for advertising a company's digital services.

    There don't seem to be too many large companies that use Materialize.

Flask

  • favorite_borderAlive and Kickin'
    Flask is pretty popular amongst Python users. In 2018, Flask was voted most popular Python web framework in the Python Developers Survey 2018. However, Django, Ruby on Rails, and other web frameworks continuously rank about Flask.
  • extensionAll Around Reliable
    Flask is easy to pick up and use for simple projects. It doesn't take too many lines of code to do what you want to do, so your codebase may be smaller than, for example, Django. Database integration is easy, and it has high scalability.
  • settingsNot Meant for Big Things?
    There seems to be some discourse in the community where some say that Flask is not suited for big projects. However, there are others that claim that this is a myth.

    There's also no login functionality, admin site, and no default web template engines. You must build everything yourself.
  • publicMr.Worldwide
    Some sites that utilize Flask are Pinterest, LinkedIn, Reddit, Uber, Netflix, and Airbnb. Uses include everything from social media apps, blogs, and streaming services.
